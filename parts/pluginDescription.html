Generate Markdown documents based on the interface


GitHub | Issues | Website | LICENSE

English introduction

The Markdown document is generated based on the interface and can be previewed, edited, and uploaded to YApi or ShowDoc.

  • Controller/Dubbo interface to generate Markdown documents
  • Supports viewing Markdown source code, previewing Markdown, editing interface fields, etc.
  • Support custom configuration Markdown template
  • Support exporting Markdown files to local
  • Support uploading interface documents to YApi,ShowDoc,Yuque
  • Generate Http Client

中文介绍

基于接口生成 Markdown 文档, 支持预览、编辑以及上传到YApi、ShowDoc、语雀。

  • Controller/Dubbo 接口生成Markdown文档

  • 支持自定义配置 Markdown 模版
  • 支持导出 Markdown 文件到本地
  • 支持上传到 YApi、ShowDoc、语雀
